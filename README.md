# PSL: Formatted
A project with the aim of providing the Public Suffix List in different formats.

## Pulling the formatted files
You can pull the formatted versions of the PSL from the [`**list**`](https://github.com/wdhdev/psl-formatted/tree/main/list) folder in this repository!

## FAQ
### How can I trust the formatted versions of the PSL?
We are fully open-source. You can see the different formatting scripts [here](https://github.com/wdhdev/psl-min/tree/main/scripts).

### How often are the formatted versions updated?
All formats are automatically updated every 24 hours via GitHub Actions and are automatically written to the `list` folder.
